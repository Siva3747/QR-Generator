# 📱 QR Code Generator Web App

A simple and interactive QR Code Generator built using HTML, CSS, and JavaScript.
This project allows users to generate QR codes instantly for any text or URL with a clean and responsive user interface.

---

# 🚀 Features

* Generate QR codes instantly
* User-friendly interface
* Responsive design
* Supports text and URL input
* Dynamic QR code display

---

# 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* QR Code API

---

# 📂 Project Structure

Qr-Code-Generator/
│── index.html
│── style.css
│── script.js

---

# ⚙️ How It Works

* Enter any text or URL in the input field
* Click the Generate button
* The QR code will be created instantly
* Users can scan the QR code using any mobile scanner app

---

# 💡 Key Concepts

* DOM Manipulation
* Event Handling
* API Integration
* Dynamic Image Generation
* Responsive Web Design

---

# 📌 How to Run the Project

1. Clone or download this repository
2. Open `index.html` in any web browser
3. Enter text or a URL and generate the QR code
